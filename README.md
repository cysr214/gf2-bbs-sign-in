# gf2-bbs-sign-in

少前2社区每日签到、每日任务、自动兑换

## Secrets / Variables / ENV

- `BBS_USERNAME` (必须)  
  用户名
- `BBS_PASSWORD` (必须)  
  密码
- `AUTO_EXCHANGE` (可选)  
  自动兑换道具，按优先级顺序排列，以逗号 `,` 分隔，默认为 `day,month,life,*`
  - 可以是兑换周期，如 `day,month,life` 分别表示“每日限购”、“每月限购”、“终身限购”
  - 可以是物品名称，只要名称包含就算，如 `情报拼图,萨狄斯金,战场报告,解析图纸,基原信息核,火控校准芯片,共键,次世代内存条`
  - `*` 表示任意物品，可以放在最后以防出现其他未考虑到的物品
- `FAILED_WEBHOOK` (可选)  
  失败时会以 GET 方式请求

## 使用构建产物

你可以在 Release 下载构建好的程序来运行

此时可以在程序同目录下创建 `.env` 文件，在里面配置上述环境变量

```dotenv
BBS_USERNAME=xxxx
BBS_PASSWORD=xxxx
```

### 定时运行注意事项

为了便于在定时任务等场景下执行签到，程序有以下特性：

- 会在工作目录输出日志 `gf2-bbs-sign-in.log`，该日志只包含最近一次运行的日志，不会越堆越多
- 在 Windows 下直接运行不显示命令提示符窗口

你可以利用 Windows 的任务计划、Linux 的 crontab 等来每日定时运行，但需注意：

- `.env` 的读取和输出日志文件均基于**工作目录**，因此你需要注意使用正确的工作目录，如
  - Linux：`cd` 到对应目录执行，而不是直接使用绝对路径执行
  - Windows：任务计划中相关运行配置需要配置正确的“起始路径”
